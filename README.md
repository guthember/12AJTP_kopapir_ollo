# Kõ - papír - olló játék

## Játékszabály
- Kõ    legyõzi az ollót
- Papír legyõzi a  követ
- Olló  legyõzi a  papírt

## Játékmenet
- számítógép gondol az egyikre (véletlenszám)
- a felhasználó megadja a saját lehetõségét
  - (K)õ, (P)apír, (O)lló
- eredmény kiértékelése
- eredmény kiírása
- 3 nyert játsza -> játék megnyerése
